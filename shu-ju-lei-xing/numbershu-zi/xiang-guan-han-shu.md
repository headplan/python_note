# ceil(x):返回数字的上入整数

# exp(x):返回e的x次幂(e^x)

# fabs(x):返回数字的绝对值

# floor(x):返回数字的下舍整数

# log(x):返回数字的对数

# log10(x):返回以10为基数的x的对数

# max(x1,x2,...):返回给定参数的最大值,参数可以为序列

# min(x1,x2,...):返回给定参数的最小值,参数可以为序列
