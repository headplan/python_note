# Python增强提案

#### 什么是PEP

PEP 是 Python 增强提案\(Python Enhancement Proposal\)的缩写 . 社区通过PEP来给 Python 语言建言献策 , 每个版本你所看到的新特性和一些变化都是通过PEP提案经过社区决策层讨论、投票决议，最终才有我们看到的功能 .

[https://www.python.org/dev/peps/](https://www.python.org/dev/peps/)
