# PlanBlog

#### 搭建开发环境

```
python3 --version
Python 3.5.1
# 安装异步框架aiohttp
pip3 install aiohttp
# 安装前端模板引擎jinja2
pip3 install jinja2
# 安装MySQL的Python异步驱动程序aiomysql
pip3 install aiomysql

# 升级pip
pip3 install --upgrade pip
```

**目录结构**

```
backup/ - 备份目录
conf/ - 配置文件
dist/ - 打包目录
www/ - Web目录
 - static/ - 静态文件
 - templates/ - 模板文件

# mkdir backup/ conf/ dist/ www/ www/static www/templates
```
