# 切片

对于取指定索引范围的操作 , Python提供了切片\(Slice\)操作符 . 

```
L = ['Michael', 'Sarah', 'Tracy', 'Bob', 'Jack']
# 取前三个元素
L[0:3]
# 第一个索引为0 , 可以省略
L[:3]
# 延伸
L[1:3]
L[-2:-1] # 负数是倒着来的
# 超出范围全取
L[:10]

L = list(range(100))
L[:10] # 来10个
L[-10:] # 后10个
L[10:20] # 前11-20个
L[:10:2] # 前10个数，每两个取一个 [0, 2, 4, 6, 8]
L[::5] # 所有数，每5个取一个
L[:] # 原样输出

tuple也是一种list
(0, 1, 2, 3, 4, 5)[:3]

字符串'xxx'也可以看成是一种list
'ABCDEFG'[:3]
'ABCDEFG'[::2]
```
